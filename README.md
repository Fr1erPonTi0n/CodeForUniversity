# CodeForUniversity
Так называется мой репозитории, который создан для отправки кода (отчёт за работу) моих преподователей.
Я буду выкладывать свой различные программы, коды и т.д по дисциплинам! Сам проект переведён на систему управления зависимостями uv, что позволяет легко запускать его со всеми необходимыми библиотеками с нужной версией Python и использовать другие преимущества uv.  ‿︵‿ヽ(°□° )ノ︵‿︵
# scr/
Папка со всеми заданиями университета. Пронумерованные папки соответствуют годам учёбы. Внутри каждой папки находятся дисциплины, где хранятся отчёты за соответствующий год. В некоторых папках с дисциплинами есть подпапки 1sem, 2sem, где хранятся отчёты за первое или второе полугодие. В отдельной папке "Other" будут сохраняться различные скрипты, которые сделаны по фану. Структура этой папки будет произвольной и не станет соответствовать какому-либо шаблону.   o(≧▽≦)o
# Как будут оформлены работы?
- Если это лабораторная, то она будет создана по такому шаблону: "laborant_n", где n - это номер лабораторной работы. (ﾉ´ヮ`)ﾉ*: ･ﾟ
- Некоторые лабораторные работы оформлены в виде папки, внутри которой находятся ответы на задания. Они создаются по шаблону task_n. Также в этой папке может быть главный исполняемый файл main.py и сопутствующие ему файлы (например, text.txt). Кроме того, в зависимости от задания, может быть предусмотрено несколько вариантов запуска. (^///^)
- В некоторых дисциплинах требуется создание проекта, папка с проектом обычно называется "project". ╚(•⌂•)╝
- Остальные шаблоны будут появявяться со временем. Если это случиться, то репозиторий будет изменён. ( ‾́ ◡ ‾́ )
